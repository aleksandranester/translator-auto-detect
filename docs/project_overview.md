# Обзор проекта
Проект «Интеллектуальный переводчик с автоматическим определением языка» разрабатывается как универсальное средство перевода текстовой информации. Основной акцент сделан на удобстве пользователя и автоматизации процесса определения языка.
Программа ориентирована на использование в веб-, мобильных и настольных приложениях и может быть расширена дополнительными модулями.
