Приложение с автоопределением языка, голосовым вводом и распознаванием текста с изображений. Разработано на Python в рамках дисциплины ПППИ.

✏️

↓
🔍

↓
🌐

Архитектура
Модули проекта
10 компонентов
01 Главный модуль main.py ▶
Точка входа в приложение. Инициализирует все компоненты системы и запускает основной рабочий процесс: определение языка, перевод, кэш, история.
02 Интерфейс пользователя ui.py ▶
Управляет полем ввода исходного текста, выбором целевого языка и отображением результата перевода. Адаптивный макет для разных устройств.
03 Определение языка language_detection.py ▶
Автоматически анализирует символьный состав текста и определяет исходный язык. Пользователю не нужно указывать язык вручную.
04 Движок перевода translation_engine.py ▶
Основной функциональный модуль. Выполняет перевод текста на указанный целевой язык через внешний API или встроенные алгоритмы.
05 История переводов history_manager.py ▶
Сохраняет пары «исходный текст — перевод» в течение сессии. Позволяет повторно просматривать и использовать ранее выполненные переводы.
06 Распознавание OCR ocr_module.py ▶
Извлекает текст из графических файлов форматов PNG, JPEG, BMP и передаёт его в движок перевода. В реальной реализации использует Tesseract.
07 Голосовой ввод voice_input.py ▶
Захватывает аудио с микрофона и преобразует речь пользователя в текстовую строку для последующего перевода.
08 Настройки пользователя settings.py ▶
Хранит персональные параметры приложения: язык перевода по умолчанию и тема оформления интерфейса (светлая / тёмная).
09 Оптимизация производительности optimization.py ▶
Кэширует результаты переводов для снижения нагрузки на внешний API и ускорения повторных запросов к одним и тем же текстам.
10 Локализация интерфейса localization.py ▶
Обеспечивает поддержку нескольких языков интерфейса приложения. Хранит строки на разных языках и возвращает нужную по ключу.
